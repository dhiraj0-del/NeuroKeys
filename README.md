# NeuroKeys

> Type Anywhere. Adapt Everywhere.

NeuroKeys is an AI-powered touchless keyboard that enables users to type on any flat surface using only a webcam and computer vision.

---

## Features

- AI Hand Detection
- Finger Tracking
- Motion Tracking
- Modular Architecture
- Real-time Computer Vision
- Future Gesture Recognition
- Adaptive Virtual Keyboard

---

## Tech Stack

- Python 3.12
- OpenCV
- MediaPipe
- NumPy
