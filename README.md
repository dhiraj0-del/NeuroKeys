# 🧠 NeuroKeys

> **Type Anywhere. Adapt Everywhere.**

NeuroKeys is an AI-powered touchless virtual keyboard that enables users to type on any flat surface using only a webcam and computer vision.

Built as a modular AI project, NeuroKeys combines real-time hand tracking, motion analysis, and intelligent interaction to create a natural typing experience without physical hardware.

## ✨ Features

- Real-time webcam input
- MediaPipe AI hand detection
- 21 hand landmark tracking
- Index fingertip tracking
- Motion tracking (dx, dy)
- Modular architecture
- Open-source development
